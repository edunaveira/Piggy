Ayuda

El juego consiste en recoger el mayor número de monedas posibles antes de finalizar las 3 vidas de las que se dispone. Para ello tenemos que mover al cerdito de un lado al otro, para que la moneda no se escape.

Para controlar el juego podemos utilizar el acelerómetro, inclinando nuestro teléfono para realizar el movimiento, utilizar la pantalla táctil para moverlo o en caso de disponer de teclado, podemos utilizar los controles de dirección(flechas de dirección izquierda y derecha) para realizar el movimiento.
